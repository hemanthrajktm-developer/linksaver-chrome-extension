## How to Use

1. **Save Links**: Click the extension icon or use Ctrl+Shift+S
2. **Organize**: Create folders and add tags to categorize your links
3. **Search**: Use the search bar to quickly find any saved link
4. **Manage**: Edit, delete, or move links between folders
5. **Backup**: Export your data regularly to keep it safe

## Privacy & Security

- All data stored locally in your browser
- No external servers or cloud storage
- No tracking or analytics
- Complete data portability with export/import

## Keyboard Shortcuts

- `Ctrl+Shift+S` (Windows) / `Cmd+Shift+S` (Mac) - Quick save current page
- `Ctrl+Shift+L` (Windows) / `Cmd+Shift+L` (Mac) - Open dashboard
- `Ctrl+K` / `Cmd+K` - Focus search (in dashboard)
