# LinkSaver Installation Guide

## Quick Setup (5 minutes)

### 1. Download the Extension
- Download or clone this repository to your computer
- Extract the files if downloaded as ZIP

### 2. Install in Chrome
1. Open Google Chrome
2. Go to `chrome://extensions/`
3. Turn ON "Developer mode" (toggle in top-right)
4. Click "Load unpacked"
5. Select the `linkvault-extension` folder
6. The LinkSaver extension should now appear in your extensions list

### 3. Pin the Extension
1. Click the puzzle piece icon in Chrome toolbar
2. Find "LinkSaver" in the list
3. Click the pin icon to pin it to your toolbar

### 4. Test the Extension
1. Open the `test-extension.html` file in Chrome
2. Click the LinkSaver icon in your toolbar
3. Try saving the current page
4. Test keyboard shortcuts:
   - `Cmd+Shift+S` (Mac) / `Ctrl+Shift+S` (Windows) - Save page
   - `Cmd+Shift+L` (Mac) / `Ctrl+Shift+L` (Windows) - Open dashboard

## Troubleshooting

### Extension not showing up?
- Make sure "Developer mode" is enabled
- Refresh the extensions page
- Check that you selected the correct folder

### Keyboard shortcuts not working?
1. Go to `chrome://extensions/shortcuts`
2. Find LinkSaver
3. Set your preferred key combinations

### Dashboard showing empty?
- The extension includes test data on first load
- Try saving a few links first
- Check browser console for any errors

### Still having issues?
- Open Chrome DevTools (F12)
- Check the Console tab for error messages
- Make sure all files are in the correct locations
