# LinkSaver Privacy Policy

LinkSaver is a browser extension designed to help users save, organize, and manage web links locally.

## Data Collection
LinkSaver does not collect, transmit, sell, or share any personal or sensitive user data.

The extension only stores:
- URLs and page titles explicitly saved by the user
- User-created folders, tags, and preferences

All data is stored locally using the browser’s storage APIs.

## Permissions
- activeTab: Used only when the user clicks “Add Link” to read the current tab’s URL and title.
- storage: Used to save links and settings locally.

## Data Sharing
LinkSaver does not share data with third parties.
No analytics, tracking, or advertising services are used.

## Remote Code
LinkSaver does not use remote code. All code is bundled within the extension.
